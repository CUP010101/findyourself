I: 내향, E:외향
S: 감각, N:직관
T:사고, F:감정
J:판단, P:인식

I vs E
1. 힘들었던 한 주, 드디어 주말이다! => 친구들이랑 게임 한 판 하자!(E)/ 그동안 못 했던 게임, 나 혼자서 플레이해 보자!(I)
2. 친구가 재밌다고 같이 하자는 게임, 내 취향은 아닌데... => 우리 다른 게임 하자...(I) / 해 보면 의외로 재밌지 않을까? 같이 하자.(E)
3. 게임 정보를 얻고 공유하기 위해 커뮤니티에 가입하려고 하는데, 가입 조건이 왤케 어렵냐? => 귀찮아, 가입 안 할래.(I) / 까다롭긴 해도 정보가 많으니까 기다리지 뭐...(E)

S vs N
4. 게임을 구매하기 전 => 트레일러, 플레이 영상, 리뷰까지 꼼꼼히 보고 구매(N) / 이 게임은 분명 재미있을 거다. 리뷰 따윈 필요 없어.(S)
5. 새로 나온 게임을 처음 플레이할 때 => 튜토리얼부터 차근차근 보면서 플레이(N) / 튜토리얼은 겁쟁이나 보는 거다!(S)
6. 추리 게임을 플레이할 때 => 내가 추리 게임 얼마나 했는데... 저놈이 범인이야.(S) / 되게 의심스러운데 아직 증거가 없네. 일단 계속 지켜 보자.(N)

T vs F
7. 여태까지 함께 지내던 동료 캐릭터가 좀비에 물렸다 => 치료제를 찾을 때까지 당분간 함께 다닌다(F) / 우리 그룹 사람들이 위험하니 어쩔 수 없이 처리한다(T)
8. 쟤 때문에 자꾸 지는 것 같은데... => 다음엔 더 잘할 수 있다고 기운을 차려 준다(F) / 공략, 잘 하는 법 등을 가르쳐 줘서 다시는 이런 일이 없도록 만든다(T)
9. 너무나도 멋지고 예쁜 이 캐릭터, 성능이 눈물날 정도로 처참한데... => 못 써먹을 정도로 구리니 버린다(T) / 나의 최애는 도저히 포기할 수 없다(F)

J vs P
10. 내 컴퓨터 바탕 화면의 상태는? => 수많은 아이콘들로 빼곡히 찬 바탕 화면(P) / 있을 것만 있는 깔끔한 바탕 화면(J)
11. 게임을 하기 전 => 도전과제? 그게 뭐임?(P) / 도전과제 클리어가 게임의 핵심! 절대 놓치지 말자.(J)
12. 게임을 할 때 => 그때그때 내키는 게임들을 플레이한다(P) / 어떤 게임을 언제 클리어할지 계획을 정해 둔다(J)

직업:
ENTJ : 네크로멘서
ENTP : 드루이드
ENFJ : 법사
ENFP : 소환사
ESTJ : 해적
ESTP : 가디언
ESFJ : 바드
ESFP : 버서커
INTJ : 전략가
INTP : 레인저
INFJ : 마검사
INFP : 사제
ISTJ : 군주
ISTP : 연금술사
ISFJ : 클레릭
ISFP : 거상

0 = E / 1 = I / 2 = N / 3 = S / 4 = F / 5 = T / 6 = P / 7 = J

필요없음:
x 5. 내가 하고 있는 게임, 누가 자꾸 지켜 보는 것 같은데... => 맘껏 지켜 봐. 내가 어떤 사람인지 보여주겠어!(N) / 그만 쳐다보면 안 되나... 원래 잘 하는데 자꾸 안 되네...(S)
x 8. 보상으로 얻은 방어력 0.01% 상승 아이템을 장착하려면 잠깐 로비에 다녀와야 하는 상황 => 고작 0.01인데 그냥 포기하고 친구들한테 가자(E) / 잠깐 나가서 장착하고 돌아온다(I)
